# 이커머스 이미지 제작 에이전트

농수산물 **상세페이지** + **썸네일**을 자동 제작하는 Claude Code 에이전트입니다.

## 실행

```
/detail-page
```

시작 시 "상세페이지 / 썸네일" 중 선택합니다.

## 상세페이지 — 3가지 모드

| 모드 | 이미지 | 분석 | 토큰 | 추천 |
|------|--------|------|------|------|
| ⚡ 빠른 제작 (경량) | 5장 | 스킵 | 적음 | 초보자, 빠른 결과 |
| 📦 표준 제작 (중간) | 7장 | 간단 1개 | 보통 | 일반 추천 |
| 🔬 정밀 제작 (풀) | 9장+ | 병렬 3개 | 많음 | 고품질 필요 시 |
| 📋 프롬프트 추출 | 0장 | 스킵 | 최소 | 프롬프트 확인/수정 |

## 썸네일 — 2가지 타입

| 타입 | 설명 | 산출물 |
|------|------|--------|
| 🤍 클린 상품컷 | 흰 배경 + 상품만 | 3장 변형 |
| 🎯 마케팅 썸네일 | 상품 + 카피 + 뱃지 | 3장 변형 |

## 구조

- `.claude/commands/detail-page.md` — 메인 스킬
- `scripts/gemini-image.py` — Gemini 이미지 생성 (플랫폼별 규격)
- `templates/detail-base.html` — HTML 템플릿
- `prompts/` — 4가지 스타일 + 16개 블록 프롬프트
- `output/` — 생성 결과물
