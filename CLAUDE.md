# 상세페이지 AI 제작 에이전트

이커머스 농수산물 상세페이지를 자동 제작하는 Claude Code 에이전트입니다.

## 실행

```
/detail-page
```

## 2가지 모드

| 모드 | 이미지 | 분석 | 토큰 | 추천 |
|------|--------|------|------|------|
| ⚡ 빠른 제작 (경량) | 5장 | 스킵 | 적음 | 초보자, 빠른 결과 |
| 🔬 정밀 제작 (풀) | 9장+ | 병렬 3개 | 많음 | 고품질 필요 시 |

## 구조

- `.claude/commands/detail-page.md` — 메인 스킬
- `scripts/gemini-image.py` — Gemini 이미지 생성 (플랫폼별 규격)
- `templates/detail-base.html` — HTML 템플릿
- `prompts/` — 4가지 스타일 + 16개 블록 프롬프트
- `output/` — 생성 결과물
